# CareerPilot AI

An AI-powered Resume & Job Description Analyzer built with Flask.

## Features

- Upload Resume (PDF/Image)
- Upload Job Description
- ATS Match Score
- Missing Skills Detection
- Resume Skill Extraction
- Interview Readiness Rating
- Personalized Suggestions

## Tech Stack

- Python
- Flask
- HTML
- CSS
- PDFPlumber
- PyTesseract
- Git
- GitHub
- Render
